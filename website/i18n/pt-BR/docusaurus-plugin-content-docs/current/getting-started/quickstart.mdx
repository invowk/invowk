# Início Rápido

Vamos executar seu primeiro comando Invowk™ em menos de 2 minutos. Sério, pegue um café antes se quiser - você terá tempo de sobra.

## Crie Seu Primeiro Invowkfile

Navegue até qualquer diretório de projeto e inicialize um invowkfile:

<Snippet id="quickstart/init" />

Isso cria um `invowkfile.cue` com comandos iniciais para um projeto típico. Vamos dar uma olhada:

<Snippet id="quickstart/default-invowkfile" />

O template padrão inclui três comandos prontos para uso: `build`, `test` e `clean`. Não se preocupe em entender cada detalhe ainda - vamos cobrir isso em breve!

## Experimente o Runtime Virtual

Um dos superpoderes do Invowk é o **runtime virtual** - um interpretador de shell integrado que funciona da mesma forma em todas as plataformas:

<Snippet id="quickstart/virtual-runtime" />

O runtime virtual usa o interpretador [mvdan/sh](https://github.com/mvdan/sh), proporcionando comportamento consistente de shell POSIX em todas as plataformas.

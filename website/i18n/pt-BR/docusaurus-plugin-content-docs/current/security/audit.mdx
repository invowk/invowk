### Script Checker

Analisa conteĂşdo e caminhos de scripts em busca de padrĂľes de execuĂ§ĂŁo perigosos.

## DetecĂ§ĂŁo de AmeaĂ§as Compostas

O correlacionador identifica quando achados de diferentes checkers aparecem na mesma superfĂ­cie de ataque, indicando ameaĂ§as coordenadas:

| AmeaĂ§a Composta | Checkers Envolvidos | Severidade |
|-----------------|---------------------|------------|
| **ExfiltraĂ§ĂŁo de credenciais** | Env + Network | CrĂ­tica |
| **Escape path + symlink** | Script + Symlink | CrĂ­tica |
| **ExfiltraĂ§ĂŁo ofuscada** | Script + Network | CrĂ­tica |
| **Fraqueza na cadeia de confianĂ§a** | Module Metadata + Lock File | Alta |

- 3+ categorias distintas na mesma superfĂ­cie &rarr; **CrĂ­tica**
- Alta + qualquer outro achado na mesma superfĂ­cie &rarr; **CrĂ­tica**
- 2+ achados MĂŠdios na mesma superfĂ­cie &rarr; **Alta**

### Como Funciona

O checker LLM:

# Filter and File

Search and file selection components.

## Filter

Fuzzy filter through a list of options.

### Basic Usage

<Snippet id="tui/filter-basic" />

### Options

| Option | Description |
|--------|-------------|
| `--title` | Filter prompt |
| `--placeholder` | Search placeholder |
| `--width` | Filter width |
| `--height` | List height |
| `--limit` | Max selections (default: 1) |
| `--no-limit` | Unlimited selections |
| `--reverse` | Display list in reverse order |
| `--fuzzy` | Enable fuzzy matching |

## File

File and directory picker with navigation.

### Basic Usage

<Snippet id="tui/file-basic" />

### Options

| Option | Description |
|--------|-------------|
| `--title` | Title displayed above the picker |
| `--description` | Description below the title |
| `--directory` | Only show directories |
| `--file` | Only show files |
| `--hidden` | Show hidden files |
| `--allowed` | Filter by extensions |
| `--height` | Picker height |

### Examples

<Snippet id="tui/file-examples" />

### Navigation

In the file picker:
- **Up/Down**: Navigate
- **Enter**: Select file or enter directory
- **Backspace**: Go to parent directory
- **Esc/Ctrl+C**: Cancel
